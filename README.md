# SpringBoot 쇼핑몰 프로젝트

스프링부트를 사용하여 만든 쇼핑몰입니다.

## 개발 환경
* Java 11 (JDK 11.0.17)
* Server : Apache Tomcat 9.0
* Framework : Springboot, JPA
* Database: MySQL
* IDE : IntelliJ

## 사용 기술
### [Back-End]
* JAVA 11
* SpringBoot
* Spring Data JPA
* MySQL

### [Front-End]
* Thymeleaf
* HTML/CSS
* JavaScript
* jQuery
