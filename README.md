# SpringBoot 마켓 프로젝트

스프링부트로 제작한 애플리케이션 마켓 프로젝트 입니다.
Creative Market

## 개발 환경
* Java 11
* Server : Apache Tomcat 9.0
* Framework : Springboot, JPA
* Database: MySQL
* IDE : IntelliJ

## 사용 기술
### [Back-End]
* JAVA 11
* SpringBoot
* Spring Data JPA
* MariaDB

### [Front-End]
* Thymeleaf
* HTML/CSS
* JavaScript
* jQuery
